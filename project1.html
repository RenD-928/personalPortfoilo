Project Details

This project transforms a 2D painting into an interactive 3D environment using A-Frame, a web framework for building virtual reality (VR) experiences directly in the browser. Inspired by the artwork of Piet Mondrian, a renowned artist famous for his abstract geometric compositions, the recreated scene captures the essence of his signature style while immersing users in a virtual world.

The final product demonstrates how artistic expression can evolve in the digital age, combining the elegance of classic art with the dynamic nature of 3D design. By leveraging modern web technologies, this project bridges the gap between traditional creativity and emerging digital tools.

Demo Link: https://noorh13.github.io/a-frame-lmc-project/

Technologies Used

  • A-Frame: For creating and rendering the 3D environment.
  • HTML: For structuring the 3D scene and elements.
  • CSS: For styling the interface and enhancing the visual presentation.
  • JavaScript: For adding interactivity and controlling 3D elements (if applicable).
  • Generative AI (DALL-E): Used to conceptualize elements or expand on artistic designs.
  • GitHub: For version control and collaboration.

Artwork

Artist: Piet Mondrian
Painting Recreated: An abstract geometric composition from Mondrian's "De Stijl" movement, known for its simplicity and use of primary colors alongside black and white.
