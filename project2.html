Chatbot: Walter White Experience

Creating an interactive chatbot inspired by Walter White from *Breaking Bad*

  • Back to Home

Project Details

The Walter.AI Chatbot is designed to simulate conversations with the iconic character Walter White from the television series *Breaking Bad*. By training the bot on lines of dialogue from the series, it accurately imitates Walter's speech patterns and personality, creating a realistic and immersive experience.

The chatbot allows users to interact with Walter White in-character, responding to user prompts in a way that reflects his distinct personality and behavior. Users can explore hypothetical conversations or scenarios not directly shown in the series, enhancing the interactive storytelling experience.

Demo Link: https://aayushanantha-code.github.io/ChatBot/

Technologies Used

  • OpenAI GPT Framework: For generating realistic and in-character responses.
  • HTML: For structuring the chatbot interface.
  • CSS: For styling the interface to match Breaking Bad's aesthetics (green and black color palette).
  • JavaScript: For handling chatbot interactions and functionality.
  • 3D Avatar Rendering: A custom-made 3D render of Walter White for the visual component.
  • API Key Validation: Ensuring proper user input of API keys to maintain security.
  • GitHub: For version control and collaboration.
